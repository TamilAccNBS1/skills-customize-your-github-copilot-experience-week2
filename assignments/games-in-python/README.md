
# 📘 Assignment: Hangman Game Challenge

## 🎯 Objective

Build the classic word-guessing game using Python strings, loops, conditionals, and random selection. You'll create a text-based game where players guess letters to reveal a hidden word before running out of attempts.

## 📝 Tasks

### 🛠️ Build the Hangman Game

#### Description
Create a complete Hangman game that manages word selection, player guesses, and game state tracking. Players should be able to make letter guesses and see their progress toward solving the puzzle.

#### Requirements
Completed program should:

- Randomly select words from a predefined list
- Accept letter guesses and display current progress in underscore format (_ _ _)
- Track incorrect guesses remaining and display them to the player
- End the game when the word is guessed or attempts are exhausted
- Display appropriate win/lose messages with final game state
